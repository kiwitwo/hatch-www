# Hatch-www
Hatch-www is the main repository for Hatch's frontend web interface. You can access the latest deployment at [hatch.lol](https://hatch.lol/). It is deployed automatically via Cloudflare Workers.
## Running locally
Clone the repository:
```
git clone https://github.com/hatchdotlol/hatch-www
cd hatch-www
```
Make sure you have [Node.js](https://nodejs.org/en) installed, then install dependencies.
```
npm install
```
Finally, run the dev server.
```
npm run dev
```
Now, make some changes and you'll be able to view and test them at the URL given by the dev server.
